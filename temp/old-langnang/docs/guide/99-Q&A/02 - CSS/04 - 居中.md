## 水平居中

### 行内元素

给其父元素设置 text-align:center,即可实现行内元素水平居中

```css
.parent {
	text-align: center;
}
```

### 块级元素

该元素设置 margin:0 auto 即可

```css
.self {
	margin: 0 auto;
}
```

### 子元素包含`float`属性

为了让子元素水平居中, 则可让父元素宽度设置为 fit-content,并且配合 margin

```css
.parent {
	width: -moz-fit-content;
	width: -webkit-fit-content;
	width: fit-content;
	margin: 0 auto;
}
```
