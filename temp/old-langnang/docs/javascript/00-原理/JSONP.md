# JSONP

## 原理

### 中文或英文

JSON with Padding

### 用途

可以让网页从别的域名（网站）那获取资料，即跨域读取数据。

通过 script 不受同源策略限制来达到跨域的目的

### 核心概念或运作流程

技术核心是前端构造 script 发起 get 请求，后端将数据放到 js 回调里，前端接受响应后执行回调拿到数据

### 代码书写思路

### 技术优点

通过简单的约定就能跨域

### 技术缺点

不支持 get 以外的动词，而且存在 csrf 风险

### 弥补缺点

解决办法是 CORS 或 csrf token
