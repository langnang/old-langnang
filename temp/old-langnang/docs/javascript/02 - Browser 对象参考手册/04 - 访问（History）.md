# History

> History 对象包含用户（在浏览器窗口中）访问过的 URL。

## History 对象属性

| 属性   | 描述                              |
| ------ | --------------------------------- |
| length | 返回浏览器历史列表中的 URL 数量。 |

## History 对象方法

| 方法      | 描述                                |
| --------- | ----------------------------------- |
| back()    | 加载 history 列表中的前一个 URL。   |
| forward() | 加载 history 列表中的下一个 URL。   |
| go()      | 加载 history 列表中的某个具体页面。 |
