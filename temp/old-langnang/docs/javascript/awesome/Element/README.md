> A Vue.js 2.0 UI Toolkit for Web.

## Components

**basic**

- Layout 布局
- Container 布局容器
- Color 色彩
- Typography 字体
- Border 边框
- Icon 图标
- Button 按钮
- Link 文字链接

**Form**

- Radio 单选框
- Checkbox 多选框
- Input 输入框
- InputNumber 计数器
- Select 选择器
- Cascader 级联选择器
- Switch 开关
- Slider 滑块

- Rate 评分

- Tag 标签

- Tree 树形控件
- Pagination 分页
- Badge 标记

- Message 消息提示

- Breadcrumb 面包屑
