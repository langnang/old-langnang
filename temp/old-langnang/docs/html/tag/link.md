# link - HTML Tag

`<link>` 标签定义文档与外部资源的关系。

`<link>` 标签最常见的用途是链接样式表。

## 实际应用

### 链接外部样式表

```html
<link href="style.css" type="text/css" />
```

### 定制页面图标（即页面标签前边的小图标）

```html
<link rel="shortcut icon" href="favicon.ico" type="image/x-icon" />
<!-- 注释：href="favicon.ico" 这里边放这个图标的图形文件地址。 -->
```
