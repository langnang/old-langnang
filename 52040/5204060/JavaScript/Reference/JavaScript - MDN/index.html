# JavaScript | MDN

# Excerpt

JavaScript (JS) is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. While it is most well-known as the scripting language for Web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB and Adobe Acrobat. JavaScript is a prototype-based, multi-paradigm, single-threaded, dynamic language, supporting object-oriented, imperative, and declarative (e.g. functional programming) styles. Read more about JavaScript.

JavaScript（JS）是一种具有函数优先 (opens new window)特性的轻量级、解释型或者说即时编译型 (opens new window)的编程语言。虽然作为 Web 页面中的脚本语言被人所熟知，但是它也被用到了很多非浏览器环境 (opens new window)中，例如 Node.js (opens new window)、Apache CouchDB (opens new window)、Adobe Acrobat (opens new window) 等。进一步说，JavaScript 是一种基于原型 (opens new window)、多范式、单线程的动态语言，并且支持面向对象、命令式和声明式（如函数式编程）风格。了解更多关于 JavaScript (opens new window)。

本部分文档将专注于 JavaScript 语言本身，而非局限于网页或其他运行环境。想要了解网页中特有的 API (opens new window)，请参考 Web API (opens new window) 以及 DOM (opens new window)。

ECMAScript 语言规范（ECMAScript Language Specification） (opens new window)（ECMA-262）和ECMAScript 国际化 API 规范（ECMAScript Internationalization API specification） (opens new window)（ECMA-402）是 Javascript 的标准。当某个 ECMAScript 新特性的提案 (opens new window)已经被一些浏览器实现时，MDN 上的文档或示例就可能会涉及到这些新特性。大多数时候，处在 stages (opens new window) 3 和 4 的新特性会被收录到文档中，且收录时间通常早于其正式发布的时间。

不要将 JavaScript 与 Java 编程语言 (opens new window)混淆。虽然“Java”和“JavaScript”都是 Oracle 公司在美国和其他国家注册（或未注册）的商标，但是这两门语言在语法、语义与用途方面有相当大的不同。
